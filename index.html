Flex Wrap

A propriedade CSS flex-wrap define se os itens flexíveis são solicitados a ficar na mesma linha ou podem ser quebrados em várias linhas. Se o argumento for válido, ele define uma direção em que as linhas são empilhadas.

Nowrap

Os itens flexíveis são agrupados em uma linha única, o que pode fazer com que o flex container transborde. O cross-start é equivalente ao início ou antes , dependendo do valor da flex-direction. Este é o valor padrão.
